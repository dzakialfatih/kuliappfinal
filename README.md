# 📱 KuliApp - Labor Service

KuliApp adalah aplikasi Android untuk mencari dan menyewa jasa kuli renovasi rumah dengan mudah. 
Aplikasi ini memungkinkan pengguna menemukan pekerja berdasarkan rating dan lokasi, serta menghubungi mereka langsung melalui WhatsApp.
Permintaan terhadap jasa penggunaan tenaga kerja informal seperti kuli terus meningkat. Namun, masyarakat masih menghadapi kesulitan dalam menemukan kuli 
yang terpercaya, cepat tersedia, serta memiliki informasi harga dan keahlian yang jelas. 
Di sisi lain, penyedia jasa kuli masih bergantung pada koneksi pribadi untuk memperoleh pekerjaan. Penelitian ini bertujuan untuk merancang dan mengembangkan 
aplikasi sewa jasa kuli berbasis Android dengan pendekatan Design Thinking guna menghasilkan solusi digital yang efisien dan mudah digunakan. 
Proses perancangan dilakukan melalui lima tahapan, yaitu Empathize, Define, Ideate, Prototype, dan Testing.

## 🚀 Fitur Utama
- 🔍 Pencarian kuli berdasarkan rating
- 📍 Pencarian kuli berdasarkan nama lokasi
- ⭐ Sistem rating pekerja
- 💬 Pemesanan langsung via WhatsApp
- 👤 Profil pekerja

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- REST API
- Material Design UI
